# AI-Based-Automatic-Modulation-Recognition-DVB-S2X
Dataset

The dataset used in this project is DeepSig RadioML 2018.01A.

Download Link
https://www.deepsig.ai/datasets

Dataset File
GOLD_XYZ_OSC.0001_1024.hdf5

This project presents an AI-based Automatic Modulation Recognition (AMR) system for DVB-S2X satellite communication using Deep Learning. Two deep learning models, namely Convolutional Neural Network (CNN) , Bidirectional Long Short-Term Memory (BiLSTM) are developed and compared for modulation classification using DeepSig RadioML 2018.01A dataset.
